# 371. Sum of Two Integers

Given two integers `a` and `b`, return _the sum of the two integers without using the operators_ `+` _and_ `-`.

**Example 1:**

<pre>
**Input:** a = 1, b = 2
**Output:** 3
</pre>

**Example 2:**

<pre>
**Input:** a = 2, b = 3
**Output:** 5
</pre>

**Constraints:**

* `-1000 <= a, b <= 1000`

Accepted `297.3K` Submissions `587.3K`
